This is a full-stack Online Casino application featuring a slot game and secure user authentication system. The project demonstrates both backend and frontend development with a focus on performance, security, and interactive UI.

Tech Stack :
Backend
C# with .NET
SQL Server
Entity Framework (Migrations)
Frontend
React
TypeScript
PixiJS (for slot game rendering)

Authentication & Security :

Implemented a multi-layer authentication system
Passwords are secured using hashing and salting
Designed to ensure safe user data handling and protection

Features :

Interactive slot machine game built with PixiJS
Smooth animations and responsive gameplay
User authentication (login/register)
Backend API integration with frontend
Database migrations using Entity Framework


Developed as a portfolio project to demonstrate full-stack development skills.
